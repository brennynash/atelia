üöÄ Deployment Commands

# Install Firebase CLI (if not installed)
npm install -g firebase-tools

# Login to Firebase
firebase login

# Initialize (if needed)
firebase init

# Deploy everything
firebase deploy

# Deploy only hosting
firebase deploy --only hosting

# Deploy only Firestore rules
firebase deploy --only firestore:rules

# Deploy only Storage rules
firebase deploy --only storage:rules

‚öôÔ∏è Local Development Setup

# Start local server (Python)
python -m http.server 8000

# Or use the batch file
start-server.bat

# Then open browser to:
http://localhost:8000

# For Firebase emulators (optional)
firebase emulators:start

‚ö†Ô∏è Important Security Notes

  • Change Admin Password: The default password is public. Change it immediately in Firebase Console.
  • Cloudinary Upload Preset: Make sure "atelia_uploads" preset is created in Cloudinary (unsigned).
  • Firestore Rules: Already deployed and secured. Only admin@ateliabuilt.com can edit content.
  • Environment Variables: All configs are in code files. For production, consider using environment variables.
  • Backup Data: Regularly export Firestore data from Firebase Console.
